Fink - Dokumentation

Dies ist eine Sammlung verschiedener Dokumente für Fink. Einige der Dokumente sind auch dann interessant, wenn man Fink nicht nutzt, aber lernen möchte, wie man Unix-Software nach Mac OS X oder Darwin portiert.

Nutzer-Dokumentation

Die aktuelle Nutzer-Dokumentation für Fink:

  • Fink-Benutzerhandbuch - Hier wird beschrieben, wie man Fink und Pakete installiert und auf eine neue Version von Fink aktualisiert. Das Handbuch behandelt sowohl die Quelltext- als auch die Binär-Distribution.
  • Fortgeschrittene Fink Themen - beschreibt weiter fortgeschrittene Konzepte als die des Benutzerhandbuchs.
  • X11 auf Darwin und Mac OS X laufen lassen - Hier werden die Konzepte, die Installation und der Start von X11 beschrieben (auch generell für Darwin und Mac OS X Nutzer)

Ein Bündel von Dokumenten, die zwar vollständiger, aber auch mehr oder weniger veraltet sind und nicht mehr gepflegt werden:

  • Installation und Aktualisierung - Wie installiert man Fink und wie aktualisiert man auf eine neue Version
  • Nutzung - Wie benutzt man Fink und die installierte Software
  • Fink Lies-Mich - Die Lies-Mich Datei für die Quelltext-Distribution
  • Git Zugang - Wie greift man auf das Git-Repository von Fink zu, wenn man zwischen bestimmten Versionen von Fink die neuesten Quelltext-Pakete haben möchte.

Entwickler-Dokumentationen

  • Handbuch über die Sicherheitspolitik - Eine Pflichtlektüre für alle, die selbst Pakete in Fink pflegen oder neu erstellen wollen.
  • Using Fink: A Developer's How To (2MB pdf file) - Folien eines Vortrags auf der O'Reilly Mac OS X Conference (auch als PowerPoint Folien erhältlich)
  • Porting Tipps - Notizen für die Portierung von Unix Programmen nach Darwin
  • Paket erstellen - Tutorial - eine Ergänzung zum Handbuch für die Paketerstellung, die auf sich auf reale Beispiele konzentriert und eine Einführung über das Paketerstellen für Anfänger bietet.
  • Handbuch für die Paketerstellung - Wie erstellt und pflegt man Pakete für Fink
  • Das Fink Entwickler Wiki - enthält auch Material für Entwickler, das noch in Bearbeitung ist.

Andere Dokumente

  • Anleitung für die Internationalisierung - Material für die fortlaufende Arbeiten, um die Webseiten zu internationalisieren.
  • Mailing List Netiquette - Wie setzt man die Fink Mailing-Listen am besten ein.
