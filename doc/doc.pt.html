Fink - Documentação

Este é um conjunto de vários documentos escritos para o Fink. Alguns destes documentos também podem ser úteis a pessoas que usam o Mac OS X sem o Fink mas que desejam aprender a portar software Unix.

Documentação do usuário

Documentação do usuário atualizada:

  • Guia do usuário do Fink - cobre a instalação do Fink, instalação de pacotes e atualização para uma nova versão do Fink. Contém instruções para ambas as versões binária e código fonte.
  • Guia de tópicos avançados do Fink - cobre conceitos mais avançados do que os cobertos no Guia do usuário.
  • Executando o X11 no Mac OS X - cobre conceitos, instalação e execução (também direcionado a usuários do Mac OS X em geral).

Alguns documentos que são mais completos mas estão um pouco desatualizados e não são mais mantidos:

  • Installation and Upgrading - como instalar o Fink ou atualizá-lo para uma versão nova (em inglês)
  • Usage - como usar o Fink e os softwares instalados (em inglês)
  • Leia-me do Fink - o Leia-me da distribuição de códigos fontes
  • Acesso via Git - como acessar o repositório git do Fink para obter os fontes dos pacotes mais recentes antes da liberação de uma nova versão

Documentação do desenvolvedor

  • Manual de política de segurança - leitura obrigatória para todos os que mantêm pacotes no Fink ou gostariam de adicionar seus próprios pacotes.
  • Using Fink: A Developer's How To (arquivo PDF com 2MB) - slides de uma apresentação na O'Reilly Mac OS X Conference (também disponível como um arquivo PowerPoint) (em inglês)
  • Porting Tips - notas para os interessados em portar aplicações Unix para o Mac OS X (em inglês)
  • Packaging Tutorial - um complemento ao Manual de empacotamento que foca exemplos reais e faz uma introdução sobre empacotamento para iniciantes. (em inglês)
  • Packaging Manual - como criar e manter pacotes no Fink (em inglês)
  • The Fink Developer Wiki - material relacionado a desenvolvimento que ainda está em construção. (em inglês)

Outros documentos

  • Internationalization Guide - material sobre o esforço em andamento de internacionalização do site (em inglês)
  • Mailing List Netiquette - como melhor usar as listas de discussão do Fink. (em inglês)
