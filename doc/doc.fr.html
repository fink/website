Fink - Documentation

Voici un ensemble de documents écrits pour Fink. Certains de ces documents peuvent également être utiles aux personnes qui ne se servent pas de Fink sous Mac OS X ou Darwin, mais souhaitent apprendre à porter des logiciels Unix sur ces systèmes.

Documentation utilisateur

Documentation actuelle de l'utilisateur de Fink :

  • Guide de l'utilisateur de Fink - ce guide couvre l'installation de Fink, celle des paquets et la mise à jour de Fink lors d'une nouvelle version. Il contient les instructions pour les versions source et binaire.
  • Guide des notions complexes dans Fink - couvre des sujets plus complexes que ceux abordés dans le Guide de l'utilisateur.
  • X11 sur Darwin et Mac OS X - couvre les concepts généraux, l'installation et le lancement (les utilisateurs de Darwin et de Mac OS X y trouveront aussi des conseils utiles)

Un grand nombre de documents plus complets, mais obsolètes et non remis à jour :

  • Installation et Mise à jour - comment installer Fink et le mettre à jour lors d'une nouvelle version
  • Utilisation - comment utiliser Fink et les logiciels installés
  • Fink ReadMe - le fichier ReadMe de la distribution source
  • Accès Git - comment accéder au serveur Git de Fink pour récupérer les paquets source les plus récents entre deux versions

Documentation développeur

  • Charte de sécurité - lecture obligatoire pour tous ceux qui sont mainteneurs de paquets dans Fink ou qui veulent le devenir.
  • Utilisation de Fink : un guide pour le développeur (fichier pdf 2MB) - diapositives d'une présentation lors de la Conférence O'Reilly Mac OS X (disponible aussi sous format PowerPoint)
  • Trucs et astuces pour le portage - notes sur le portage d'applications Unix sur Darwin
  • Tutoriel d'empaquetage - complément au Guide de construction des paquets. Basé sur des exemples réels, il introduit le débutant dans le monde de la création de paquets.
  • Guide de construction des paquets - comment créer et maintenir des paquets Fink
  • Le wiki des développeurs de Fink - inclut du matériel en construction pour les développeurs.

Autres Documents
